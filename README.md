## Authors

- [@fjribptra](https://www.github.com/fjribptra)


## Installation

Install @fjribptra/hello-world package

```bash
  npm install @fjribptra/hello-world
```
    
## Usage/Examples

```javascript
const hello = require('@fjribptra/hello-world')

hello() //run hello function to print "hello world"
```



## Run the project
